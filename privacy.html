Privacy Policy

Last Updated: July 18, 2025

1. Overview

The AvantageCars application (package name: app.avantagecars.com) respects your privacy. This Privacy Policy explains how the app handles data and user information.

2. Data Collection

Our application does not collect any personal data automatically. The user can simply browse the available cars and choose a vehicle.

3. Functionality

The application allows users to:

  • View available cars for rental.
  • Select a car and send a rental request via WhatsApp to finalize the reservation.

No other data is collected, stored, or shared by the application.

4. Permissions

The application does not request any special permissions on your device.

5. Data Sharing

No data is shared with third parties. The only communication happens via WhatsApp when the user voluntarily contacts us.
